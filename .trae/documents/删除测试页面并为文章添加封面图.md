## 为文章添加封面图片

### 任务内容

1. **删除测试页面**
   - 删除 `frontend/public/article-cover-tool.html`
   - 删除 `frontend/public/debug-login.html`

2. **创建 Python 脚本**
   - 从网上下载高质量图片（使用 Unsplash API）
   - 上传图片到后端 OSS
   - 为文章随机分配封面图片

### 实现步骤

1. 创建 `backend/scripts/add_article_covers.py` 脚本
   - 使用 requests 下载图片
   - 调用 `/api/v1/images/` API 上传图片
   - 调用 `/api/v1/articles/{id}` API 更新文章封面

2. 脚本功能
   - 自动获取所有已发布的文章
   - 为每篇文章下载一张随机图片
   - 上传并设置封面
   - 支持指定图片关键词（如技术、自然、建筑等）

3. 图片来源
   - 使用 Unsplash Source API（免费，无需 API key）
   - 或者使用 picsum.photos（随机图片服务）

### 预期结果
- 删除测试页面
- 所有文章都获得封面图片
- 封面图片存储在 OSS 上
