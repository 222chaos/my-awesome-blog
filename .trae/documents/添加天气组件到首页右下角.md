## 天气组件接入方案

### 方案概述
在首页右下角添加一个 Glassmorphism 风格的天气组件，实时显示当天天气信息。

### 技术选型
**选择 OpenMeteo API**
- ✅ 完全免费，无需 API Key
- ✅ 支持地理定位（IP/经纬度）
- ✅ 数据准确，更新及时
- ✅ 支持中文语言
- ✅ 无跨域问题
- API 基础地址：`https://api.open-meteo.com/v1`

### 实现计划

#### 1. 创建天气服务 (`frontend/src/services/weatherService.ts`)
- 获取用户位置（使用 IP geolocation API）
- 调用 OpenMeteo API 获取天气数据
- 缓存机制（1小时过期）
- TypeScript 类型定义

#### 2. 创建天气组件 (`frontend/src/components/home/WeatherCard.tsx`)
- Glassmorphism 设计风格（与现有组件一致）
- 显示内容：
  - 当前温度
  - 天气状况（晴/多云/雨等）
  - 湿度和风速
  - 天气图标
- 右下角固定定位
- 响应式设计（移动端隐藏或缩小）
- 加载/错误状态处理

#### 3. 集成到首页
- 在 `page.tsx` 中导入 `WeatherCard`
- 添加到页面布局（固定右下角）
- 保持与其他组件的视觉一致性

### 数据结构
```typescript
interface WeatherData {
  current: {
    temperature: number;
    weatherCode: number;
    humidity: number;
    windSpeed: number;
  };
  location: {
    name: string;
    country: string;
  };
}
```

### 视觉设计
- 使用 `bg-glass/30 backdrop-blur-xl` 玻璃效果
- 使用 `text-white` 保证对比度
- 悬停效果：`hover:-translate-y-1`
- 图标使用 Lucide React 或 SVG
- 支持 light/dark 模式

### API 调用流程
1. 用户打开页面 → 检测位置缓存
2. 无缓存 → 调用 ipapi.co 获取 IP
3. 使用经纬度调用 OpenMeteo API
4. 缓存结果（1小时有效期）
5. 组件渲染天气数据
