# 统计卡片宽度与主内容对齐方案

## 当前布局分析

**StatsPanel 主结构：**
```tsx
<section className="py-8 sm:py-10 md:py-12 lg:py-16">
  <div className="container mx-auto px-4 sm:px-6 lg:px-8">
    <div className="grid grid-cols-1 lg:grid-cols-12 gap-6 lg:gap-8">
      <div className="lg:col-span-4 flex flex-col gap-6">
        <ProfileCard />
        <FriendLinks />
      </div>
      <div className="lg:col-span-8 flex flex-col gap-6">
        <ArticleList />
        <StatsCharts />
      </div>
    </div>
  </div>
</section>
```

**StatsCharts 当前布局：**
```tsx
<div className="grid grid-cols-1 md:grid-cols-2 gap-6">
  <Card>月度统计</Card>
  <Card>周活跃度</Card>
</div>
```

## 问题

StatsCharts 使用自己的 `grid` 布局，可能导致卡片宽度与父容器（`lg:col-span-8`）不完全对齐。

## 解决方案

**方案 1：保持当前布局（推荐）**
- StatsCharts 继续使用 `grid grid-cols-1 md:grid-cols-2 gap-6`
- 由于它在 `lg:col-span-8` 容器内，宽度已经由父容器控制
- 不需要额外修改

**方案 2：使用 Flex 布局**
- 改为 `flex flex-col md:flex-row gap-6`
- 让两个卡片平分容器宽度

## 建议

保持当前的 `grid` 布局即可，因为：
1. 父容器 `lg:col-span-8` 已经控制了宽度
2. `md:grid-cols-2` 在中等屏幕上会平分可用空间
3. 整个结构已经有 `container mx-auto px-4 sm:px-6 lg:px-8` 确保全局对齐

当前实现已经正确，不需要修改。
