# 修复相册页面顶部区域字体颜色

## 问题分析

当前相册页面顶部区域的文字颜色问题：
- 标题 "我的相册" 使用 `text-tech-cyan` - 在浅色模式下可能在渐变背景上不够明显
- 描述 "探索生活中的美好瞬间，用镜头记录难忘时刻" 使用主题默认色
- 统计信息 "72 张照片 / 6 个相册" 使用 `text-sm` 没有明确颜色类

## 修复方案

**AlbumsPage.tsx 顶部区域修复**

1. **标题修复**
   - 从 `text-tech-cyan` 改为 `themedClasses.textClass`
   - 这样在深色模式下显示为 `text-foreground`（白色）
   - 在浅色模式下显示为 `text-gray-800`（深灰色）

2. **统计信息修复**
   - 添加 `themedClasses.mutedTextClass`
   - 深色模式：`text-foreground/70`（半透明白色）
   - 浅色模式：`text-gray-600`（灰色）

## 修复效果

修复后，在浅色模式下：
- 标题将显示为深灰色，与渐变背景形成良好对比
- 统计信息显示为中等灰色，清晰可读
- 在深色模式下保持白色/半透明白色
