## 启动步骤

### 1. 检查后端服务状态
- 检查 terminal 11 中的后端服务是否正常运行
- 确认后端是否已在 http://127.0.0.1:8989 上运行

### 2. 启动前端服务
- 进入 frontend 目录
- 运行 `npm run dev` 启动 Next.js 开发服务器
- 前端将在 http://localhost:3000 上运行

### 3. 验证服务
- 确认前端可以访问后端 API（http://127.0.0.1:8989）
- 打开浏览器访问 http://localhost:3000 查看前端页面

**注意事项：**
- 后端端口固定为 8989（根据 api.ts 配置）
- 如果后端未运行，需要先启动后端：`cd backend && .venv\Scripts\activate && uvicorn app.main:app --reload --port 8989`
- 前端会自动连接到后端 API
