## 留言板 UI/UX 升级计划：Cyberpunk 风格

我将把留言页面打造成一个大胆前卫的、赛博朋克风格的“数字涂鸦墙”。

### 1. 设计风格 (Cyberpunk UI)
- **核心概念**：霓虹 (Neon)、故障艺术 (Glitch)、HUD 界面、深色模式。
- **配色方案**：
  - 背景：深邃蓝黑 (`#0F0F23`)
  - 主色调：电光紫 (`#7C3AED`)
  - 强调色：霓虹粉 (`#F43F5E`)、青色 (`#00D9FF`)
- **排版**：
  - 标题：**Syne** (艺术感、前卫)
  - 正文：**Manrope** (现代、易读)
- **布局**：**Bento Grid** (便当盒网格)，将弹幕、留言列表、输入框和统计信息模块化展示。

### 2. 后端接口扩展
为了支持新的 UI 展示，需要在 `backend` 增加以下公开接口：
- `GET /api/v1/messages/trending`: 获取热门留言（按点赞数排序），用于展示“高光时刻”。
- `GET /api/v1/messages/stats/activity`: 获取近7天的留言活跃度趋势，用于 HUD 统计图表。

### 3. 前端实现步骤
1.  **字体引入**：在 `layout.tsx` 中引入 Google Fonts (Syne, Manrope)。
2.  **组件开发**：
    - `GlitchText`: 带有故障抖动效果的标题组件。
    - `NeonCard`: 带有发光边框和玻璃拟态背景的容器组件。
    - `CyberButton`: 切角设计、带有 hover 动画的按钮。
3.  **服务更新**：更新 `messageService.ts` 适配新接口。
4.  **页面重构** (`messages/page.tsx`)：
    - 使用 CSS Grid 实现 Bento 布局。
    - 整合全屏弹幕背景。
    - 制作沉浸式的输入体验。

### 4. 预期效果
页面将不再是一个简单的列表，而是一个动态的、交互性强的数字空间，用户仿佛在未来的数字终端上留下印记。
