## 添加文章封面图片上传功能

### 后端
后端已有完整的 API 支持，无需修改：
- `POST /api/v1/images/` - 上传图片
- `PUT /api/v1/articles/{article_id}` - 更新文章（支持 cover_image 字段）

### 前端需要添加的功能

1. **在 `articleService.ts` 中添加以下方法**：
   - `uploadImage(file: File)` - 上传图片并返回 URL
   - `updateArticle(id: string, data: Partial<Article>)` - 更新文章
   - `setArticleCover(articleId: string, file: File)` - 便捷方法：上传图片并设置为封面

2. **创建测试工具页面**：
   - 类似 debug-login.html
   - 允许选择文章 ID 和上传图片
   - 显示上传结果

### 预期结果
- 可以通过前端上传图片并设置为文章封面
- 后端图片存储在 OSS
- 文章封面正确显示
