# 删除并重新设计相册标题组件

## 任务分析

需要删除的组件：
```tsx
<motion.div 
  initial={{ opacity: 0, y: -20 }} 
  animate={{ opacity: 1, y: 0 }} 
  transition={{ duration: 0.5 }} 
  className="text-center mb-12" 
> 
  <h1 className="text-4xl md:text-5xl font-bold mb-4 flex items-center justify-center text-white"> 
    <Camera className="mr-3" /> 我的相册 
  </h1> 
  <p className="text-xl max-w-2xl mx-auto text-white/90"> 
    探索生活中的美好瞬间，用镜头记录难忘时刻 
  </p> 
  <div className="flex items-center justify-center gap-4 mt-6 text-sm text-white/70"> 
    ...统计信息 
  </div> 
</motion.div>
```

## 新组件设计方案

**现代风格标题组件特点：**
1. **符合主题** - 使用 `themedClasses` 支持深浅色切换
2. **玻璃态设计** - 使用 `bg-glass/30 backdrop-blur-xl`
3. **渐变效果** - 标题文字使用渐变色
4. **装饰元素** - 添加现代感的装饰性元素
5. **动画效果** - 保持 framer-motion 动画但更精致
6. **统计卡片** - 将统计信息改为独立的小卡片

## 具体实施

1. **删除原组件**：直接替换为新设计
2. **新组件结构**：
   - 玻璃态卡片容器
   - 渐变标题（tech-cyan 渐变）
   - 副标题带装饰线
   - 统计信息改为两个玻璃态小卡片
   - 添加装饰性背景元素
