# 相册页面现代化 UI/UX 优化方案

基于您的需求（大胆前卫、丰富组件、参考开源），我为您设计了一套 **"Cyber-Gallery (赛博画廊)"** 风格的优化方案。

我们将参考 **Aceternity UI** 和 **Magic UI** 等现代开源组件库的设计理念，使用 `Framer Motion` 和 `GSAP` 实现高性能动画。

## 1. 设计理念：沉浸式数字艺术馆
- **视觉风格**：深色模式为主，搭配霓虹光晕（Tech Cyan），强调**玻璃拟态**与**3D 空间感**。
- **交互核心**：滚动视差（Parallax）、鼠标跟随（Cursor Follow）、焦点强调（Focus Effect）。
- **布局策略**：打破传统网格，使用**瀑布流 (Masonry)** 和 **Bento Grid (便当盒布局)**。

## 2. 新增核心组件 (Rich Components)

### A. `ParallaxScrollHero` (视差滚动英雄区)
- **功能**：页面顶部的视觉冲击区。
- **效果**：三列图片以不同速度向上滚动，营造深邃的空间感。
- **技术**：`framer-motion` 的 `useScroll` 和 `useTransform`。

### B. `FocusCards` (聚光灯卡片组)
- **功能**：展示相册列表。
- **效果**：当鼠标悬停在某张卡片上时，其他卡片变暗/模糊，当前卡片高亮并轻微上浮，聚焦用户注意力。
- **参考**：Vercel / Aceternity UI 的卡片交互。

### C. `GlitchText` (故障风标题)
- **功能**：用于页面主标题 "MY ALBUMS"。
- **效果**：随机的像素偏移和颜色分离效果，符合"前卫"的科技感设定。

### D. `MasonryPhotoGrid` (智能瀑布流)
- **功能**：相册详情内的图片展示。
- **效果**：图片加载时的交错上浮动画（Staggered Fade In），点击图片实现**共享元素转场 (LayoutId)** 放大查看。

## 3. 页面结构改造 (Page Restructure)

我们将重写 `src/app/albums/page.tsx`，结构如下：

1.  **Hero Section**: 
    - 背景：动态粒子或网格。
    - 内容：`GlitchText` 标题 + 简短介绍。
    - 视觉：`ParallaxScrollHero` 作为背景层或侧边装饰。

2.  **Featured Albums (Bento Grid)**:
    - 使用不规则网格展示精选相册，打破单调。

3.  **All Albums (Focus Cards)**:
    - 沉浸式的卡片列表，带有强交互感。

4.  **Interaction Layer**:
    - 添加自定义鼠标光标（可选），悬停在图片上时变为"查看"图标。
