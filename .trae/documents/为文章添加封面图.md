## 为文章添加封面图方案

### 方案概述
通过网络搜索获取图片，通过后端API上传到阿里云OSS，然后将图片URL更新到文章的cover_image字段。

### 执行步骤

1. **检查OSS配置状态**
   - 查看后端启动日志，确认OSS是否可用
   - 如果OSS未配置，使用外部图床URL作为替代方案

2. **创建封面图管理脚本**
   - 文件位置：`backend/scripts/add_article_covers.py`
   - 功能：
     - 从网上下载高质量技术类封面图
     - 通过API上传图片到OSS
     - 更新文章的cover_image字段

3. **图片选择策略**
   为6篇文章选择匹配主题的封面图：
   - FastAPI异步编程：Python/代码主题
   - React Server Components：React/前端主题
   - PostgreSQL全文搜索：数据库主题
   - Docker多阶段构建：容器/DevOps主题
   - AI大模型应用开发：AI/科技主题
   - TypeScript高级类型：编程/代码主题

4. **执行脚本并验证**
   - 运行脚本上传图片
   - 验证文章的cover_image字段已更新
   - 前端查看效果

### 技术细节
- **图片上传接口**：`POST /api/v1/images/`
- **文章更新接口**：`PUT /api/v1/articles/{article_id}`
- **图片要求**：JPG/PNG格式，建议尺寸 800x400
- **认证方式**：Bearer Token

### 备选方案
如果OSS未配置：
- 使用Unsplash、Pexels等免费图床的图片URL
- 直接更新文章的cover_image字段为外部URL
