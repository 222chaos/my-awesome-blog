# 文章页面 UI/UX 深度优化方案

根据用户需求（"modern", "bold", "avant-garde", "rich components"）和 `ui-ux-pro-max` 的分析，我们将进一步升级文章页面，打造一个具有**瑞士现代主义 2.0 (Swiss Modernism 2.0)** 风格的赛博朋克体验。

## 1. 核心设计语言升级
-   **排版 (Typography)**: 引入 `Syne` (标题) 和 `Manrope` (正文) 字体组合，营造前卫时尚感。
-   **布局 (Layout)**: 强化网格系统 (Grid System)，使用不对称布局增加视觉张力。
-   **色彩 (Colors)**: 保持深色基调 (#18181B)，使用高饱和度的霓虹粉 (#EC4899) 作为强调色，对比强烈的黑白灰阶。

## 2. 新增丰富组件 (Rich Components)

### A. 头部：故障艺术标题 (Glitch Header)
-   使用现有的 `GlitchText` 组件替换标准标题。
-   添加动态副标题，使用打字机效果。

### B. 精选区：聚光灯卡片 (Focus Featured)
-   使用 `FocusCards` 组件替换现有的 `FeaturedCarousel`（或者作为另一种视图选项）。
-   当鼠标悬停在卡片上时，卡片高亮，周围变暗，创造沉浸式聚焦体验。

### C. 列表区：智能瀑布流 (Smart Masonry)
-   集成 `MasonryGrid` 组件，实现非规则的瀑布流布局。
-   卡片大小动态变化（1x1, 1x2, 2x1, 2x2），打破单调的网格。

### D. 视觉增强：视差滚动背景 (Parallax Background)
-   使用 `ParallaxScroll` 的逻辑，为背景添加缓慢移动的几何图形或粒子，增加深度感。

### E. 交互细节 (Micro-interactions)
-   **Magnetic Buttons**: 按钮对鼠标有轻微的磁吸效果。
-   **Custom Cursor**: 自定义鼠标光标，悬停在可点击元素上时变形。

## 3. 实施步骤

1.  **引入新字体**: 配置 Next.js Font (Syne & Manrope)。
2.  **集成 GlitchText**: 在 `ArticlesPage` 顶部应用故障文字效果。
3.  **集成 MasonryGrid**: 替换现有的 CSS Grid 布局，使用 `MasonryGrid` 组件展示文章列表。
4.  **集成 FocusCards**: 在精选区域使用 `FocusCards` 展示 Top 3 文章。
5.  **优化 CommandBar**: 调整样式以匹配新的霓虹粉强调色。
6.  **全局样式调整**: 更新背景色和文字颜色，符合 Swiss Modernism 2.0 规范。
