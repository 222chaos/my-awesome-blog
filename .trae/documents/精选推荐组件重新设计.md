# 精选推荐组件重新设计方案

## 现状分析

当前首页有两个精选推荐相关组件：
1. **FeaturedHighlights** - 轮播式推荐，显示4个静态项目
2. **FeaturedSection** - 3列网格，但实际只使用4个项目

## 新设计方案

### 设计目标
- 从轮播式改为网格卡片式，提高内容可见度
- 增加视觉层次和交互效果
- 整合分类、标签、热度等信息
- 保持 glassmorphism 设计风格和科技主题

### 新组件特性

#### 1. 多维度分类展示
- **置顶推荐** (Pinned) - 重要内容，突出显示
- **热门精选** (Trending) - 高热度内容
- **最新发布** (Latest) - 最新内容
- **深度专题** (Featured) - 专题系列

#### 2. 卡片设计元素
- 渐变图标背景
- 悬停缩放和光晕效果
- 分类标签和热度指示器
- 流畅的进入动画

#### 3. 响应式布局
- 移动端：2列网格
- 平板：3列网格
- 桌面：4列网格

#### 4. 视觉增强
- 动态渐变背景
- 悬浮动画效果
- 统计数据展示（浏览量、点赞数）
- 优雅的过渡动画

## 实施步骤

1. **重新设计 FeaturedHighlights 组件**
   - 移除轮播逻辑
   - 改为网格卡片布局
   - 添加更多视觉元素和交互效果
   - 增强响应式设计

2. **移除 FeaturedSection 组件**
   - 该组件与新的设计重复
   - 从 page.tsx 中移除引用

3. **优化数据结构**
   - 扩展 HighlightItem 接口，添加更多字段
   - 支持统计数据、分类标签等

4. **更新样式和动画**
   - 使用 framer-motion 添加流畅动画
   - 增强玻璃态效果
   - 添加悬停交互反馈

## 技术要点

- 保持现有 tech-cyan、tech-sky 等主题色
- 使用 GlassCard 组件保持一致性
- 应用 animate-fade-in-up 等预定义动画
- 响应式设计覆盖所有屏幕尺寸
