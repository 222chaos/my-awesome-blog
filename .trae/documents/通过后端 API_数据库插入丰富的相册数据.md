# 通过后端 API 接口插入丰富的相册数据

## 当前情况
- 后端已有 6 个测试相册（城市夜景、自然风光、人物肖像、旅行记忆、动物世界、美食摄影）
- 创建相册的 API 端点：`POST /api/v1/portfolio/`（需要超级用户权限）

## 实施方案

### 方案一：使用 API 接口（推荐）
1. **创建或使用管理员账户**
   - 检查数据库中是否已有超级用户
   - 如果没有，创建一个管理员账户

2. **获取访问令牌**
   - 使用管理员账户登录
   - 获取 JWT 访问令牌

3. **通过 API 创建相册**
   - 使用令牌调用 `/api/v1/portfolio/` 端点
   - 批量插入丰富的相册数据

### 方案二：直接数据库插入（更快速）
1. **扩展现有的测试数据脚本**
   - 创建更多样化的相册主题
   - 包含不同风格的摄影作品

2. **数据主题包括**
   - 建筑摄影
   - 街头摄影
   - 微距摄影
   - 黑白摄影
   - 时尚摄影
   - 运动摄影
   - 抽象摄影
   - 航空摄影
   - 静物摄影
   - 水下摄影

## 推荐实施
使用**方案二**（直接数据库插入），因为：
- 更快速，无需处理认证
- 更可靠，避免网络问题
- 适合一次性批量导入数据

## 数据集设计
创建 10-15 个精选相册，每个包含：
- 标题、描述（中文）
- Unsplash 高质量封面图
- 适当的分类标签（technologies 字段）
- 合理的日期范围
- 精选标志（约 30-40% 为精选）

## 执行步骤
1. 创建增强的测试数据脚本
2. 运行脚本插入数据
3. 验证数据正确显示在前端
