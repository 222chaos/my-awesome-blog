## 新增弹幕数据计划

### 目标
创建一个新脚本来为留言板添加更多有趣的弹幕数据，使其更生动。

### 步骤

1. **创建脚本文件** `backend/scripts/seed_danmaku.py`
   - 使用现有的 User 模型和 Message 模型
   - 连接数据库获取测试用户

2. **设计弹幕内容**
   - 创建30-40条符合 Cyberpunk 主题的弹幕
   - 包含技术梗、流行文化、有趣评论
   - 使用不同的颜色（青色、粉色、黄色、绿色、紫色、红色）
   - 混合弹幕和普通留言类型

3. **实现功能**
   - 使用不同的用户ID创建弹幕
   - 设置合适的颜色值
   - 随机分配 likes 数量
   - 添加各种科技感的内容

### 示例弹幕内容
- "这个UI设计太酷了！🎮"
- "FastAPI + React = 完美组合 💻"
- "赛博朋克风格爱了 😍"
- "弹幕效果很有意思 ✨"
- "代码质量很高 👍"
- "期待更多文章 📚"
- "技术栈选得真好 🚀"

### 预期结果
脚本运行后，留言页面将显示更多丰富的弹幕内容，增加页面活力。
