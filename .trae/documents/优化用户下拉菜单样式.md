# 用户下拉菜单样式优化方案

## 当前状态分析

**UserProfileMenu.tsx** 包含以下特征：
- 使用 Radix UI DropdownMenu 基础组件
- 通过 `useThemedClasses` 提供主题样式
- 使用 framer-motion 实现菜单项交错动画
- 基础样式较为简单，缺乏视觉层次

## 优化目标

### 1. 增强玻璃态效果
- 使用项目的 glassmorphism 设计系统
- 更明显的背景模糊效果
- 优化边框和透明度

### 2. 改进菜单项交互
- 更明显的悬停效果
- 图标动画增强
- 更流畅的颜色过渡

### 3. 优化视觉效果
- 增加内边距和间距
- 改进分隔线样式
- 增强阴影和光晕效果
- 优化标签（我的账户）样式

### 4. 改进动画
- 更自然的菜单项进入动画
- 优化触发按钮的悬停效果
- 改进在线状态指示器动画

### 5. 响应式优化
- 确保在不同屏幕尺寸下的良好显示
- 优化触摸设备的交互体验

## 实施步骤

1. **优化 UserProfileMenu.tsx**
   - 改进 DropdownMenuContent 的样式
   - 增强菜单项的悬停和焦点状态
   - 优化动画参数
   - 改进标签和分隔线样式
   - 增强退出登录项的视觉反馈

2. **优化 useThemedClasses.ts**
   - 改进下拉菜单背景类
   - 增强菜单项的悬停类
   - 优化阴影效果类
   - 改进分隔线类

3. **保持设计一致性**
   - 使用 tech-cyan 主题色
   - 遵循 glassmorphism 设计系统
   - 保持与其他组件的视觉一致性
