# 优化相册标题页面 UI/UX - 设计方案 A

## 设计亮点

根据提供的设计方案 A，优化相册标题页面：

### 主标题区域特点
1. **沉浸式设计** - 动态粒子背景/渐变动画
2. **玻璃拟态** - 圆角卡片 + 背景光晕 + 边框发光
3. **动态图标** - 360° 无限旋转，带渐变背景
4. **彩虹渐变文字** - `from-tech-cyan via-tech-sky via-purple-400 to-pink-500`
5. **动态分隔线** - 中间有呼吸动画圆点
6. **打字机效果** - 副标题

### 统计信息卡片特点
1. **双卡片并排** - 照片数和相册数分开显示
2. **悬停光效** - 背景渐变 + 上浮效果
3. **旋转图标** - 8秒旋转动画，带边框
4. **数字动画** - 使用 CountUp 实现数字滚动

## 实施计划

由于项目中没有 `CountUp` 组件，将：
1. 简化实现 - 直接使用静态数字，添加简单的入场动画
2. 移除 `CountUp` 组件依赖
3. 保留所有视觉效果和动画

## 具体修改

- 替换当前的标题组件为设计方案 A 的完整实现
- 适配 `themedClasses` 主题支持
- 移除不必要的 `CountUp` 依赖
