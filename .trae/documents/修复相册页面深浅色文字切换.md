# 相册页面深浅色文字切换修复方案

## 问题分析

相册页面的多个组件使用了硬编码的文字颜色，无法根据主题切换：

### 1. **AlbumsPage.tsx** - 主要问题
- 标题 `text-white` (line 248)
- 描述 `text-tech-lightcyan` (line 251)
- 统计信息 `text-white/60` (lines 254, 257, 262)
- 空状态 `text-white/60` 和 `text-white/30` (lines 334-335)

### 2. **AlbumCard.tsx** - 卡片问题
- 精选标签 `text-black` (line 107)
- 分类标签 `text-white` (line 119)
- 图片数量 `text-white/80` (line 161)
- 标题 `text-white` (line 179)
- 描述 `text-white/80` (line 187)
- 统计信息 `text-white/60` (line 195)

### 3. **AlbumFilter.tsx** - 过滤器问题
- 输入框 `text-white` 和 `placeholder:text-white/50` (line 99)
- 排序按钮文字 `text-white` (line 111)
- 排序选项 `text-white` (line 130)
- 搜索结果 `text-white/60` (line 195)

## 解决方案

### 1. 修复 AlbumsPage.tsx
- 使用 `useThemedClasses` hook
- 将 `text-white` 改为 `themedClasses.textClass`
- 将 `text-white/60` 改为 `themedClasses.mutedTextClass`
- 将 `text-white/30` 改为 `themedClasses.mutedTextClass` + 额外透明度

### 2. 修复 AlbumCard.tsx
- 添加 `useThemedClasses` hook
- 更新所有文字颜色使用主题类
- 精选标签需要特殊处理（浅色模式文字应该为白色）

### 3. 修复 AlbumFilter.tsx
- 添加 `useThemedClasses` hook
- 更新输入框、按钮和下拉菜单的文字颜色
- 排序选项使用 `themedClasses.textColorClass`

## 注意事项

- 相册页面使用了固定的渐变背景 `bg-gradient-to-br from-tech-darkblue via-tech-deepblue to-tech-cyan`
- 这可能会影响浅色模式的可读性，但根据用户要求只修复文字颜色
- 精选标签需要保持青色背景，所以文字应该始终为黑色
